# Kitchen Management System (Java)

This project is a simple object-oriented simulation of a kitchen environment developed using Java.

## Features
- Object-oriented design using multiple classes
- Kitchen item management (Refrigerator, Oven, Table)
- Position tracking system (Location class)
- Area calculation and space control
- Dynamic object storage using ArrayList

## Technologies
- Java
- Object-Oriented Programming (OOP)
- ArrayList (Collections Framework)

## Project Structure
- `Konum` → Handles object positions (x, y coordinates)
- `Buzdolabi`, `Fırın`, `Masa` → Represent kitchen items
- `Mutfak` → Manages all items and controls available space
- `Oda (Main)` → Runs the simulation

## What I Learned
- Designing systems using object-oriented principles
- Managing relationships between classes
- Working with collections (ArrayList)
- Implementing basic algorithms and validations

## How to Run
1. Open the project in NetBeans or any Java IDE
2. Run the `Oda` class
3. The program will simulate adding items to a kitchen and display results
